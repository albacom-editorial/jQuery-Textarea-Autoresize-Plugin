Autoresizing Textarea

Example usage of autoresizing <textarea>. Built using jQuery. Download plugin here: jquery.autoresize.js

Simple jQuery plugin to add vertical autoresize functionality to a textarea. Tested and working on Chrome 7, Safari 4, Firefox 3.6 and MSIE6.

default functionality:
$('#textarea1').autoresize();

setting some options:
$('#textarea2').autoresize({ animate: false, buffer: 2, onresize: function() { ... } });
